# Build Settings
Add config.js file under static/js
```
var firebase_config = {
    apiKey: "",
    authDomain: "",
    databaseURL: "",
    projectId: "",
    storageBucket: "",
    messagingSenderId: "",
    appId: ""
      };
firebase.initializeApp(firebase_config);
```
update the above values
